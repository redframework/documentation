Introduce

Red Framework is a Modern PHP, JS MVC Framework Merged with Special Tools and Components Developed by Mahbod Azizkhani (RedCoder).

After Passing a year it has Been reached to Version 1.0.0
and released in 2 Development Kits :


Enterprise SDK :

Containing all Red Framework Components to Develop Enterprise Web Applications with PHP as Back-end and JS as Front-end


Console App Development Kit :

Containing Necessary Components to Develop Console Applications with PHP as Fast as Possible

Enterprise SDK


Enterprise SDK is a full-featured and Modern MVC Framework to Speed up and Organize Development
it Contains High Quality Components such As:

Perfect Database Engine which Supports 3 Famous Databases (MySQL, SQLServer, SQLite), 3 Available Database Slot, and High-Performance ORM which Supports CRUD Operations

Captcha Service

SMTP Mail Service

Safe Upload Service

Validate, Filter, Sanitize Service

Organized Language Service

High-Quality Routing System with Middleware

Organized Inputs(URL Parameters, GET, POST Variables and Etc)

Configured And Crypted Sessions

Debug Components Such as Debugger, DebugBar and a lot More ... !


Here is Enterprise SDK Structure :


  • app
  • config
  • database
  • public
  • resources
  • routes
  • storage
  • vendor
  • views

Console App Development Kit


Console App Development Kit Contain all Necessary Component and Command System to Create Console App with PHP as Fast as Possible

Here is Console Kit Structure


  • app
  • commands
  • config
  • database
  • resources
  • storage
  • vendor

Red Analytics

Red Framework Comes with a Powerful tool Named 'Red Analytics'
Red Analytics Comes with general purposes : Speed Up the Development; Such as running Development Server, generating Controller, Model, View, Middleware ... and Analysing the Application such as Route and Command Controls
Red Analytics has been written with JAVA as a Cross Platform Application; so in any where you can create your new projects with Red Analytics
this tool has compatibility with both Development kits
